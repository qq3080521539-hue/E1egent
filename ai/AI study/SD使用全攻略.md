# SD使用全攻略

## 📚 课程概述

本笔记整理了Stable Diffusion（SD）人工智能商业应用摄影课程的内容，系统地介绍了SD的安装、使用、文生图、图生图、ControlNet、LORA训练等功能，帮助学习者掌握SD的使用方法。

- **新老版本对比**：SD新老版本的功能差异和特点

### 2. 基础概念
- **模型概念**：SD模型的基本概念和选择方法
  - **模型定义**：Stable Diffusion模型是一种深度学习模型，用于生成图像
  - **模型类型**：
    - **基础模型**：如SD 1.5、SD 2.1、SD XL等，提供基础生成能力
    - **写实模型**：如Realistic Vision、Deliberate等，擅长生成真实感图像
    - **二次元模型**：如Anything、Counterfeit等，擅长生成动漫风格图像
    - **艺术模型**：如Dreamshaper、MeinaMix等，擅长生成艺术风格图像
  - **模型选择方法**：
    - 根据要生成的图像风格选择合适的模型
    - 尝试不同模型，找到最适合自己需求的
    - 可以在多个模型之间切换，获得不同效果
  - **模型文件格式**：.safetensors（安全格式）、.ckpt等
- **VAE模型应用**：VAE模型的作用和使用方法
  - **VAE定义**：Variational Autoencoder（变分自编码器），负责图像的编码和解码
  - **VAE作用**：
    - 改善图像质量和色彩表现
    - 减少图像噪点和伪影
    - 提升图像的细节表现
  - **常见VAE模型**：
    - kl-f8-anime2.vae.pt（适合二次元风格）
    - anything-v4.0.vae.pt（通用VAE）
    - vae-ft-mse-840000-ema-pruned.safetensors（高质量VAE）
  - **VAE使用方法**：
    - 将VAE模型文件放入models/VAE文件夹
    - 在SD界面的VAE下拉菜单中选择要使用的VAE
    - 也可以在提示词中使用VAE名称指定
- **Embedding原理**：Embedding的原理和使用方法
  - **Embedding定义**：也称为Textual Inversion，是一种小型模型，用于学习特定的概念或风格
  - **Embedding原理**：
    - 通过少量图像训练，让模型学习特定的人物、物体或风格
    - 训练后的Embedding可以用一个单词触发
  - **Embedding特点**：
    - 文件很小（通常几十KB到几百KB）
    - 可以精确控制特定元素的生成
    - 适合学习特定人物、品牌、风格等
  - **Embedding使用方法**：
    - 将Embedding文件放入embeddings文件夹
    - 在提示词中直接使用Embedding的名称
    - 可以在提示词中调整Embedding的权重（如：(embedding_name:1.2)）
- **LORA操作**：LORA的操作和应用
  - **LORA定义**：Low-Rank Adaptation（低秩适应），是一种轻量级的模型微调方法
  - **LORA原理**：
    - 在不修改基础模型的情况下，通过小型网络学习特定风格或人物
    - 文件大小适中（通常几MB到几十MB）
    - 可以同时使用多个LORA
  - **LORA特点**：
    - 比完整模型小很多，节省存储空间
    - 可以精确控制风格或人物
    - 可以混合使用多个LORA
    - 每个LORA可以设置不同的权重
  - **常见LORA类型**：
    - **人物LORA**：学习特定人物的特征
    - **风格LORA**：学习特定的艺术风格
    - **服装LORA**：学习特定的服装样式
    - **场景LORA**：学习特定的场景风格
  - **LORA使用方法**：
    - 将LORA文件放入models/Lora文件夹
    - 在提示词中使用&lt;lora:lora_name:weight&gt;的格式
    - 也可以通过界面的LORA按钮选择
    - 权重范围通常在0.5-1.5之间

### 3. 文生图
- **文生图界面**：文生图界面的功能和操作方法
  - **界面介绍**：
    - 提示词输入框：输入正向提示词和负向提示词
    - 参数设置区：设置采样方法、步数、CFG等
    - 图像设置：设置图片尺寸、批次数、每批数量
    - 脚本区：选择和使用各种脚本功能
  - **文生图实操**：文生图的实际操作步骤和技巧
    - 操作步骤：
      1. 选择合适的模型
      2. 输入正向提示词（描述想要的图像）
      3. 输入负向提示词（描述不想要的元素）
      4. 设置图像尺寸（通常512x512或768x768）
      5. 设置采样方法和步数
      6. 点击生成按钮
    - 技巧：
      - 从简单的提示词开始，逐步添加细节
      - 使用括号可以调整提示词权重
      - 多次生成多个图像，选择最好的结果
- **提示词设计**：如何设计有效的提示词
  - **正向提示词**：
    - 主体描述：人物、物体、场景等
    - 风格描述：艺术风格、绘画风格等
    - 质量描述：高质量、高细节等
    - 环境描述：光线、背景等
  - **负向提示词**：
    - 质量问题：low quality、blurry、ugly等
    - 身体问题：extra limbs、bad anatomy等
    - 其他问题：text、watermark等
  - **提示词技巧**：
    - 使用逗号分隔不同的描述词
    - 使用括号调整权重：(word:1.2)增加权重，(word:0.8)减少权重
    - 使用AND连接不同的概念
- **参数调节**：文生图参数的调节方法
  - **采样方法**：
    - Euler a：快速，适合二次元风格
    - DPM++ 2M Karras：高质量，通用
    - DPM++ SDE Karras：高质量，适合写实
  - **采样步数**：
    - 20-30步：快速，质量一般
    - 30-50步：平衡速度和质量
    - 50步以上：高质量，速度慢
  - **CFG Scale**：
    - 7-9：通用范围
    - 越高越严格遵循提示词
    - 越低越有创意
  - **图像尺寸**：
    - 512x512：SD 1.5标准
    - 768x768：SD 2.1标准
    - 1024x1024：SD XL标准

### 4. 图生图
- **图生图界面**：图生图界面的功能和操作方法
  - **界面介绍**：
    - 图像上传区：上传参考图像
    - 提示词输入：与文生图相同
    - 参数设置：与文生图相同，额外有重绘幅度
  - **图生图实操**：图生图的实际操作步骤和技巧
    - 操作步骤：
      1. 上传参考图像
      2. 输入提示词
      3. 设置重绘幅度
      4. 点击生成
    - 技巧：
      - 重绘幅度0.3-0.5：保持原图结构，修改细节
      - 重绘幅度0.5-0.7：平衡原图结构和新元素
      - 重绘幅度0.7-1.0：几乎重新生成
- **垫图和融图**：垫图和融图的使用方法
  - **垫图**：
    - 使用参考图像作为构图基础
    - 设置较低的重绘幅度
    - 可以保持原图的构图，改变风格或内容
  - **融图**：
    - 两个或多个图像融合
    - 可以使用ControlNet的Reference Only等功能
    - 调整重绘幅度控制融合程度
- **高清重绘**：SD高清重绘的技巧和方法
  - **方法一：图生图放大**：
    - 先生成小图
    - 使用图生图放大，设置较大的尺寸
    - 重绘幅度0.3-0.5
  - **方法二：使用脚本**：
    - 使用Ultimate SD Upscale脚本
    - 可以设置放大倍数和重绘区域
    - 效果通常更好
  - **方法三：使用高清修复**：
    - 在文生图界面启用高清修复
    - 设置放大倍数和重绘幅度
    - 可以直接生成高清图

### 5. 高级功能
- **roop换脸**：roop换脸工具的使用方法
  - **功能介绍**：
    - 可以将图像或视频中的人脸替换
    - 支持保留表情和姿态
  - **使用方法**：
    - 安装roop插件
    - 上传源人脸图像
    - 上传目标图像或视频
    - 点击生成
  - **参数设置**：
    - 相似度设置：控制换脸的相似度
    - 性别设置：可以指定性别
- **ControlNet**：样片制作中最关键的一环--ControlNet
  - **ControlNet的基本概念和作用**：
    - ControlNet是一个插件，可以精确控制图像的生成
    - 可以通过线稿、深度图、姿势图等控制生成
    - 保持构图和结构，改变风格和内容
  - **控制SD构图线稿实操**：
    - 启用ControlNet
    - 上传参考图像
    - 选择预处理器（如Canny、Scribble等）
    - 选择模型
    - 设置权重和引导开始和结束
    - 点击生成
  - **常见ControlNet模式**：
    - Canny：边缘检测，保持线条
    - Scribble：涂鸦模式，手绘风格
    - OpenPose：人体姿势控制
    - Depth：深度图控制
    - Segmentation：分割图控制
    - Reference Only：参考图像风格迁移
  - **ControlNet报错问题的解决方法**：
    - 确保模型和预处理器匹配
    - 检查图像尺寸
    - 尝试不同的预处理器
- **LORA训练**：LORA模型的训练方法和技巧
  - **训练准备**：
    - 准备训练图像（10-20张）
    - 图像尺寸统一（512x512或768x768）
    - 准备提示词文件
  - **训练步骤**：
    - 使用Kohya's SS Trainer等工具
    - 设置训练参数
    - 开始训练
  - **训练参数**：
    - 学习率：1e-4到5e-4
    - 训练步数：几百到几千步
    - 网络维度：32-128
    - 网络Alpha：通常等于或大于网络维度
  - **训练技巧**：
    - 使用正则化图像防止过拟合
    - 定期保存检查点
    - 测试不同版本选择最好的结果

### 6. 商业应用
- **直播间展示样片制作**：如何做出直播间展示的样片
  - **制作流程**：
    1. 确定产品和场景
    2. 使用ControlNet控制构图
    3. 设置合适的提示词
    4. 生成多个候选图
    5. 选择最好的结果
  - **技巧**：
    - 使用Reference Only保持产品特征
    - 使用光影增强视觉效果
    - 注意产品细节的表现
- **AI加实拍样片制作**：手把手教你AI加实拍样片制作
  - **制作流程**：
    1. 拍摄产品实拍图
    2. 使用图生图功能
    3. 上传实拍图作为参考
    4. 设置较低的重绘幅度（0.3-0.5）
    5. 使用提示词优化场景
  - **技巧**：
    - 保持产品主体不变，只修改背景
    - 使用ControlNet的Canny保持产品轮廓
    - 可以多次迭代优化效果
- **电商案例**：服装换模特等电商应用案例
  - **服装换模特**：
    1. 准备服装和模特图
    2. 使用roop换脸功能
    3. 或者使用图生图+ControlNet
    4. 调整服装和模特的契合度
  - **产品展示**：
    1. 使用产品图作为参考
    2. 生成不同场景和角度
    3. 保持产品特征一致
  - **批量生成**：
    1. 使用批量生成功能
    2. 保存好的提示词模板
    3. 快速生成多张图

## 💡 关键观点

- **SD是一个强大的AI绘画工具**，可以应用于多个领域
- **掌握基础概念**是充分发挥SD能力的关键
- **ControlNet是控制构图的关键工具**
- **有效的提示词设计**是获得优质输出的关键
- **结合实拍和AI可以产生最佳效果**
- **持续学习和实践**是掌握SD使用方法的关键

## 📝 应用建议

1. **循序渐进**：从基础概念开始，逐步深入学习高级功能
2. **实践探索**：通过实际操作练习，积累经验
3. **工具选择**：根据具体需求选择合适的模型和工具
4. **持续优化**：不断调整和改进使用方法
5. **案例学习**：参考成功案例，借鉴经验

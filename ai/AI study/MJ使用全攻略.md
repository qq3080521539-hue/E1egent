# MJ使用全攻略

## 📚 课程概述

本笔记整理了Midjourney（MJ）人工智能商业应用摄影课程的内容，系统地介绍了MJ的使用方法、文生图、图生图、参数调节、关键词设计等功能，帮助学习者掌握MJ的使用方法。

## 🎯 核心内容

### 1. 基础入门
- **MJ导学**：Midjourney的基本概念和特点
  - **Midjourney简介**：
    - Midjourney是一个基于Discord的AI图像生成工具
    - 使用简单，通过文字描述生成高质量图像
    - 支持多种艺术风格和场景
  - **MJ特点**：
    - 操作简单，通过命令行使用
    - 生成速度快，质量高
    - 社区活跃，资源丰富
  - **MJ版本**：
    - v5.2：最新版本，功能强大
    - v6：测试版本，更加智能
    - niji：专门用于二次元风格
- **频道创建**：在Discord中创建MJ频道的方法
  - **Discord基础**：
    - Discord是一个聊天平台
    - Midjourney通过Discord使用
  - **创建步骤**：
    1. 加入Midjourney官方服务器
    2. 创建自己的服务器
    3. 添加Midjourney Bot到服务器
    4. 在自己的频道中使用
- **文生图功能详解**：MJ文生图功能的详细介绍
  - **基本命令**：
    - /imagine：生成图像的主要命令
    - 使用方法：/imagine prompt: [你的描述]
  - **工作流程**：
    1. 输入/imagine命令
    2. 添加描述词
    3. 发送消息
    4. 等待生成
    5. 查看和选择结果

### 2. 文生图
- **文生图实操**：MJ文生图的实际操作步骤和技巧
  - **操作步骤**：
    1. 在Discord中输入/imagine
    2. 添加详细的描述
    3. 可选：添加参数
    4. 发送消息
    5. MJ生成4张小图
  - **技巧**：
    - 从简单描述开始，逐步添加细节
    - 使用逗号分隔不同的描述
    - 描述越详细，效果越好
    - 多次生成，选择最好的
- **参数调节**：MJ参数的调节方法和技巧
  - **常用参数**：
    - --ar [宽:高]：设置宽高比
      - --ar 1:1 正方形
      - --ar 16:9 横向
      - --ar 9:16 纵向
    - --stylize [数值]：风格化程度
      - --s 0 最小
      - --s 750 最大
    - --q [数值]：质量
      - --q 0.25 快速
      - --q 1 标准
      - --q 2 高质量
    - --seed [数值]：随机种子
      - 相同seed可以重现相同结果
  - **参数组合**：
    - 可以同时使用多个参数
    - 参数之间用空格分隔
- **其他设置**：MJ其他设置的功能和使用方法
  - **风格参数**：
    - --style raw：更写实的风格
    - --style expressive：更有表现力
    - --style cute：可爱风格
  - **模式切换**：
    - /settings：打开设置界面
    - 可以选择默认版本和参数
- **关键词权重**：关键词权重以及调整的方法
  - **权重语法**：
    - word::[数字]：设置权重
    - 例如：cat::2 dog::1 猫的权重是狗的2倍
  - **权重技巧**：
    - 默认权重都是1
    - 可以设置小数权重
    - 权重可以强调或弱化某些元素
- **关键词互斥性**：关键词互斥性的识别和处理方法
  - **常见互斥词**：
    - 某些风格和场景可能冲突
    - 太多元素描述可能混乱
  - **处理方法**：
    - 简化描述，保留最重要的
    - 分多次生成，逐步添加
    - 调整权重平衡不同元素

### 3. 图生图
- **垫图和融图**：垫图和融图（图生图）的使用方法
  - **垫图（Image Prompt）**：
    - 在提示词前添加图片URL
    - MJ会参考图片的构图和风格
    - 可以同时使用多个图片
  - **使用方法**：
    1. 上传图片到Discord
    2. 右键图片，复制链接
    3. 在/imagine中粘贴链接
    4. 添加文字描述
  - **融图技巧**：
    - 使用--iw [数值]调整图片权重
    - --iw 2 图片权重更高
    - 可以结合文字描述
- **二次创作**："v""zoom"键的使用方法
  - **V按钮（Variations）**：
    - 点击U1-U4放大某张图
    - 放大后可以点击V1-V4生成变体
    - 变体保持风格，调整细节
  - **Zoom按钮**：
    - v5.2版本支持Zoom功能
    - Zoom Out 2x：缩小2倍
    - Zoom Out 1.5x：缩小1.5倍
    - Make Square：变为正方形
  - **Pan按钮**：
    - 平移功能
    - 可以向四个方向扩展图像
- **局部重绘**：MJ局部重绘应用的方法和技巧
  - **Vary (Region)**：
    - v5.2版本支持局部重绘
    - 放大图像后点击"Vary (Region)"
    - 选择要重绘的区域
    - 输入新的描述
  - **使用技巧**：
    - 可以精确修改特定区域
    - 保持其他部分不变
    - 适合修改细节
- **元素与创意提取**：describe功能的使用方法
  - **/describe命令**：
    - /describe [图片]：分析图片
    - MJ会生成4个描述词
    - 可以用这些描述词生成类似风格
  - **使用方法**：
    1. 输入/describe
    2. 上传或粘贴图片
    3. 发送消息
    4. MJ返回4个描述
    5. 点击按钮直接使用

### 4. 高级功能
- **AI换脸**：insight使用与思路
  - **InsightFace**：
    - 第三方工具，用于换脸
    - 可以结合MJ使用
  - **使用流程**：
    1. 用MJ生成图像
    2. 使用InsightFace换脸
    3. 调整参数优化效果
- **创意获取**：如何获取创意和灵感
  - **社区灵感**：
    - 查看Midjourney社区展示
    - 学习其他用户的作品
    - 可以参考描述词
  - **Describe反推**：
    - 用/describe分析喜欢的图片
    - 获取描述词和风格
    - 在此基础上修改
  - **混合实验**：
    - 混合不同的描述词
    - 尝试意想不到的组合
    - 多探索，多实验

## 💡 关键观点

- **MJ是一个强大的AI绘画工具**，可以应用于多个领域
- **掌握关键词设计**是充分发挥MJ能力的关键
- **参数调节对输出效果有重要影响**
- **垫图和融图可以产生更好的效果**
- **局部重绘是MJ的强大功能**
- **持续学习和实践是掌握MJ使用方法的关键**

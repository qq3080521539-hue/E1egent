### 第三步：构建核心工作流（2分钟）

#### 3.1 添加输入节点
**开始节点配置**：
```yaml
输入参数:
  - 产品名称: string (必填)
    示例: "2024新款运动跑鞋"
  
  - 产品特点: text (选填)
    示例: "轻便透气、减震回弹"
  
  - 目标受众: string (选填)
    示例: "25-35岁都市白领"
  
  - 视频风格: select (下拉选择)
    选项:
      - 商务专业风
      - 年轻活力风
      - 高端奢华风
      - 亲民实惠风
  
  - 视频时长: number (默认15秒)
```

#### 3.2 文案生成节点（大模型）
**节点配置**：
- **选择模型**：豆包大模型 / GPT-4 / Claude
- **系统Prompt**：
```
你是一位专业的电商文案撰写专家。根据用户提供的产品信息，
生成一段吸引人的短视频脚本，要求：
1. 开头3秒抓住眼球（痛点/悬念/数字）
2. 中间展示产品核心卖点（3-4个要点）
3. 结尾引导行动（限时优惠/立即购买）
4. 总时长控制在{视频时长}秒内
5. 语言风格符合{视频风格}定位
```

**用户Prompt**：
```
产品名称：{{产品名称}}
产品特点：{{产品特点}}
目标受众：{{目标受众}}
请生成视频脚本
```

**输出变量**：`video_script`

#### 3.3 分镜拆解节点（大模型）
**将完整脚本拆分为分镜头**：

系统Prompt：
```
你是一位专业的视频导演。将以下脚本拆解为具体的分镜头描述，
每个镜头包含：
- 镜头编号
- 时长（秒）
- 画面描述（场景、动作、运镜）
- 文字/字幕内容
- 建议的背景音乐风格
```

**输出格式示例**：
```json
{
  "scenes": [
    {
      "id": 1,
      "duration": 3,
      "visual": "产品360度旋转展示，特写细节",
      "text": "还在为跑步脚痛烦恼？",
      "music": "紧张悬疑转轻松",
      "camera": "慢速推近"
    }
  ]
}
```

**输出变量**：`storyboard`

#### 3.4 图片/视频素材生成节点（插件）

##### 方案A：使用Sora2生成视频片段
**插件配置**：
- **插件名称**：Sora2 Video Generator
- **输入参数**：
  ```
  prompt: {{storyboard.scenes[0].visual}}
  duration: {{storyboard.scenes[0].duration}}
  style: {{视频风格}}
  aspect_ratio: "16:9" (横版) / "9:16" (竖版)
  ```

##### 方案B：先生成图片再转视频（更稳定）
**步骤1：图片生成**
- **插件**：Stable Diffusion / Midjourney API
- **Prompt优化**：基于分镜描述生成详细图片提示词

**步骤2：图片转视频**
- **插件**：Runway Gen-2 / Pika Labs
- **参数**：运动幅度、镜头运动类型

#### 3.5 字幕合成节点（代码/插件）
**功能**：
- 将文字叠加到对应时间点的视频帧上
- 字幕样式：字体、颜色、位置、动画效果
- 支持自动换行和时间轴对齐

**配置示例**：
```python
# 伪代码示意
for scene in storyboard.scenes:
    add_subtitle(
        video=generated_video,
        text=scene.text,
        start_time=scene.start,
        end_time=scene.end,
        font="思源黑体 Bold",
        font_size=48,
        color="#FFFFFF",
        position="bottom_center",
        animation="fade_in_out"
    )
```

#### 3.6 背景音乐添加节点（插件）
**插件**：音频库API（如爱给网、淘声网）
- 根据视频风格自动匹配BGM
- 支持音量调节和淡入淡出
- 注意版权问题（选择免版权音乐）

#### 3.7 视频合并导出节点（结束节点）
**最终输出**：
- **输出格式**：MP4 (H.264编码)
- **分辨率**：1080x1920 (竖版) / 1920x1080 (横版)
- **帧率**：30fps
- **文件保存路径**：可自定义

## 进阶技巧

### 技巧1：Prompt工程优化

#### 文案生成Prompt模板升级
```
角色：你是拥有10年经验的{行业}顶级营销策划师
任务：为{产品名}创作爆款短视频脚本

约束条件：
- 必须包含至少3个具体数字（如"98%好评率"、"节省50%时间"）
- 前3秒必须制造冲突或悬念
- 使用"你"而非"你们"增强代入感
- 结尾必须有明确CTA（Call to Action）

参考案例结构：
[0-3s] 痛点抛出："90%的人都不知道..."
[3-8s] 解决方案展示 + 产品亮相
[8-12s] 核卖点点量化证明（对比图/数据）
[12-15s] 限时优惠 + 引导下单

输出格式：JSON（便于后续解析）
```

#### 视觉描述Prompt优化
```
基础描述：一双白色运动鞋在跑步机上
优化后：
Cinematic shot, white running sneaker on treadmill, 
studio lighting with soft shadows, 
slow motion at 120fps, 
shoe details visible (breathable mesh texture),
background blurred bokeh effect (gym environment),
color grading: warm tones, professional product photography style,
camera movement: smooth orbit around the shoe
```

## 常见问题解决

### Q1：生成的视频质量不稳定
**原因分析**：
- Prompt描述过于模糊
- 模型随机性导致结果差异
- 输入参数不一致

**解决方案**：
- 固化高质量的Prompt模板
- 设置temperature参数较低（0.3-0.7）
- 建立标准化的输入规范文档
- 使用seed值复现优质结果

### Q2：生成速度太慢
**优化方向**：
- 选择更快的模型（牺牲一定质量换取速度）
- 并行处理多个分镜（异步节点）
- 缩短单视频时长（如从30秒减到15秒）
- 缓存常用素材（如通用背景图）

### Q3：版权风险如何规避
**注意事项**：
- 使用正版授权的音乐素材库
- 避免直接使用品牌Logo（可用模糊处理）
- 人物形象建议用AI生成而非真人照片
- 文案避免夸大宣传（遵守广告法）

### Q4：如何提升转化率
**数据驱动优化**：
1. **收集数据**：记录每个视频的播放量、点赞、评论、转化
2. **分析规律**：找出高表现视频的共同特征
3. **反向优化**：将这些特征固化到工作流Prompt中
4. **持续迭代**：每周更新一次Prompt模板
